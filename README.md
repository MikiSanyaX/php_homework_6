# php_homework_6
На странице page1.php мы только устанавливаем код 404, браузер покажет свою страницу ошибки. \n
На page2.php в ссылке передан параметр text=Hello%20World, при переходе начнётся скачивание файла download.txt с содержимым "Hello World".
На page3.php сессия хранит счётчик. При каждом обновлении (или переходе по ссылке) счётчик увеличивается. Когда значение кратно трём, происходит редирект на page4.php.
На page4.php мы просто читаем счётчик из сессии и выводим его.
